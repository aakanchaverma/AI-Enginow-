# Image Classification Using Convolutional Neural Networks (CNNs)

## Overview

This project implements an Enhanced Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset.

The project demonstrates the complete deep learning workflow, including image preprocessing, data augmentation, CNN model development, training, evaluation, and visualization.

## Technologies Used

* Python
* TensorFlow/Keras
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Dataset

CIFAR-10 Dataset

Classes:

* Airplane
* Automobile
* Bird
* Cat
* Deer
* Dog
* Frog
* Horse
* Ship
* Truck

## Features Implemented

* Data Augmentation
* Batch Normalization
* Dropout
* L2 Regularization
* Early Stopping
* Confusion Matrix
* Classification Report
* Prediction Visualization

## Performance

Final Test Accuracy: ~82%
